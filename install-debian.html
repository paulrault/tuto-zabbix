Chapitre 09 Linux Installation

Debian 12 Linux

Installation complète de Debian 12 (Bookworm) dans une VM Hyper-V, partitionnement, configuration réseau statique et sécurisation de base.

01

Prérequis

ÉlémentValeur
ISO Debian 12debian-12.x.x-amd64-netinst.iso (téléchargeable sur debian.org)
RAM allouée1 024 Mo minimum (2 048 Mo recommandé)
Disque virtuel20 Go (VHDX)
Génération VMGénération 1 (meilleure compatibilité Linux)
RéseauConnexion internet recommandée pour l'installation netinstall
ℹ️

L'ISO netinstall (environ 400 Mo) télécharge les paquets pendant l'installation. Préférer l'ISO DVD (3,5 Go) si la connexion internet est limitée.

02

Démarrage sur l'ISO

Au démarrage de la VM, le menu GRUB de l'installateur Debian apparaît :

  1. Sélectionner "Graphical install" pour une installation avec interface graphique simplifiée
  2. Langue → Français
  3. Pays → France
  4. Disposition du clavier → Français
03

Configuration de base

  1. Nom de la machine → ex: debian ou srv-linux
  2. Nom de domaine → laisser vide ou entrer votre domaine (ex: home.arpa)
  3. Mot de passe root → Définir un mot de passe sécurisé pour le superutilisateur
  4. Créer un utilisateur → Nom complet, nom d'utilisateur et mot de passe
⚠️

Bien mémoriser le mot de passe root — il est indispensable pour l'administration du système. Il n'y a pas de récupération facile sur Linux.

04

Partitionnement du disque

Sélectionner "Assisté — utiliser un disque entier" pour une configuration automatique, puis :

  1. Sélectionner le disque virtuel (/dev/sda)
  2. Choisir "Tout dans une seule partition" pour une VM simple
  3. Confirmer les modifications et écrire les changements sur le disque
PartitionPoint de montageTailleType
/dev/sda1/boot512 Moext4
/dev/sda2swap2 Goswap
/dev/sda3/Resteext4
ℹ️

Pour une configuration plus avancée, choisir "Manuel" pour créer des partitions séparées pour /home, /var et /tmp.

05

Sélection des paquets

L'installateur propose une sélection de logiciels via tasksel. Pour un serveur :

OptionRecommandation
Environnement de bureau Debian❌ Décocher (inutile sur serveur)
Serveur SSH✅ Cocher (accès distant indispensable)
Utilitaires usuels du système✅ Cocher

L'installation télécharge et installe les paquets sélectionnés (5-10 minutes selon la connexion).

À la fin, confirmer l'installation de GRUB sur le disque principal (/dev/sda) puis retirer l'ISO et redémarrer.

06

Configuration réseau (IP statique)

Après le premier démarrage, se connecter en root et configurer l'IP statique :

bash
# Identifier l'interface réseau root@debian:~# ip a # Éditer la configuration réseau root@debian:~# nano /etc/network/interfaces

Contenu du fichier /etc/network/interfaces :

config
auto lo iface lo inet loopback auto eth0 iface eth0 inet static address 192.168.2.20 netmask 255.255.255.0 gateway 192.168.2.1 dns-nameservers 192.168.2.1 8.8.8.8
bash
# Appliquer la configuration root@debian:~# systemctl restart networking # Vérifier la connectivité root@debian:~# ping -c 3 8.8.8.8
07

Sécurité de base

bash
# Mettre à jour le système root@debian:~# apt update && apt upgrade -y # Installer sudo et ajouter l'utilisateur root@debian:~# apt install sudo -y root@debian:~# usermod -aG sudo votre_utilisateur # Installer fail2ban (protection contre brute-force SSH) root@debian:~# apt install fail2ban -y root@debian:~# systemctl enable fail2ban --now # Désactiver la connexion SSH en root root@debian:~# sed -i 's/PermitRootLogin yes/PermitRootLogin no/' /etc/ssh/sshd_config root@debian:~# systemctl restart ssh
✅

Debian 12 est installé et sécurisé. Vous pouvez maintenant y installer l'agent Zabbix (voir chapitre Serveur Zabbix) pour le superviser.
