Chapitre 08 Windows Client Installation

Windows 10/11

Installation de Windows 10 ou Windows 11 dans une VM Hyper-V, configuration réseau et paramètres de sécurité de base.

01

Prérequis

ÉlémentValeur recommandée
ISO Windows 10/11Téléchargeable via l'outil Media Creation Tool de Microsoft
RAM allouée2 048 Mo (Windows 10) / 4 096 Mo (Windows 11)
Disque dur virtuel40 Go minimum (VHDX)
Génération VMGénération 2 (UEFI, requis pour Windows 11)
TPM virtuelRequis pour Windows 11 — activer dans les paramètres VM Hyper-V
⚠️

Windows 11 nécessite un TPM 2.0 virtuel. Dans Hyper-V, aller dans Paramètres de la VM → Sécurité → cocher "Activer le module de plateforme sécurisée".

02

Démarrage sur l'ISO

  1. Démarrer la VM depuis le Gestionnaire Hyper-V
  2. Appuyer sur une touche rapidement pour démarrer sur l'ISO
  3. Sélectionner Langue : Français (France), Clavier : Français
  4. Cliquer sur Installer maintenant
  5. Cliquer sur "Je n'ai pas de clé de produit" si pas de licence
03

Choix de l'édition

ÉditionUsage
Windows 10/11 FamilleUsage personnel basique
Windows 10/11 Pro ✅Recommandé — accès à Hyper-V, domaine AD, GPO
Windows 10/11 EntrepriseFonctionnalités avancées en entreprise

Sélectionner Windows 10/11 Pro, accepter le contrat de licence, puis choisir "Personnalisé : installer Windows uniquement".

04

Partitionnement du disque

  1. Sélectionner le disque non alloué
  2. Cliquer sur Suivant — Windows partitionne automatiquement
PartitionTailleRôle
Récupération650 MoEnvironnement de récupération
EFI100 MoDémarrage UEFI
MSR16 MoRéservé Microsoft
Windows (C:)Reste du disqueSystème

L'installation dure environ 10-20 minutes. La VM redémarre plusieurs fois automatiquement.

05

Configuration initiale (OOBE)

Après installation, Windows lance l'assistant de configuration (Out-Of-Box Experience) :

  1. Région → France
  2. Disposition du clavier → Français
  3. Connexion → Choisir "Compte hors connexion" ou "Utilisation limitée" pour éviter l'obligation d'un compte Microsoft
  4. Nom du PC → ex: WIN10-VM ou WIN11-VM
  5. Mot de passe → Définir un mot de passe sécurisé
  6. Options de confidentialité → Désactiver les options de télémétrie inutiles
ℹ️

Sur Windows 11, Microsoft force la connexion avec un compte Microsoft. Pour contourner cela, déconnecter la VM du réseau avant l'installation, ou utiliser le nom d'utilisateur a@a.com avec un mauvais mot de passe pour forcer le mode compte local.

06

Configuration réseau (IP statique)

Configurer une IP statique via PowerShell (en administrateur) :

powershell
# Identifier l'interface réseau PS C:\> Get-NetAdapter # Configurer l'IP statique PS C:\> New-NetIPAddress -InterfaceAlias "Ethernet" ` -IPAddress 192.168.2.22 ` -PrefixLength 24 ` -DefaultGateway 192.168.2.1 # Configurer le DNS PS C:\> Set-DnsClientServerAddress -InterfaceAlias "Ethernet" ` -ServerAddresses 192.168.2.1, 8.8.8.8 # Tester la connectivité PS C:\> ping 8.8.8.8
07

Sécurité & activation

powershell
# Vérifier l'état de Windows Defender PS C:\> Get-MpComputerStatus | Select-Object AMRunningMode, RealTimeProtectionEnabled # Vérifier l'état du pare-feu PS C:\> Get-NetFirewallProfile | Select-Object Name, Enabled # Installer les mises à jour PS C:\> Install-Module PSWindowsUpdate -Force PS C:\> Get-WindowsUpdate -Install -AcceptAll -AutoReboot
✅

Windows 10/11 est prêt. Vous pouvez maintenant y installer l'agent Zabbix pour le superviser depuis votre serveur Zabbix.
