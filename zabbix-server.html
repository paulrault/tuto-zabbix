Chapitre 01 Setup

Serveur Zabbix

Installation et configuration du serveur de supervision Zabbix dans un environnement virtualisé Hyper-V sous Debian.

01

Introduction

Zabbix est une solution de supervision open source permettant de monitorer des infrastructures réseau, des serveurs et des services applicatifs. Il collecte des métriques via plusieurs protocoles : agent Zabbix, SNMP, IPMI, HTTP...

Dans ce tutoriel, le serveur Zabbix est hébergé en tant que machine virtuelle Debian sur l'hyperviseur Hyper-V domestique.

ComposantValeur
SystèmeDebian 12 (Bookworm)
Version Zabbix7.0 LTS
Adresse IP192.168.2.18
Interface webhttp://192.168.2.18/zabbix
02

Prérequis

Avant d'installer Zabbix, s'assurer que la VM Debian est à jour :

bash
root@debian:~# apt update && apt upgrade -y
ℹ️

La VM doit disposer d'au moins 2 Go de RAM et 10 Go d'espace disque pour un fonctionnement optimal.

03

Installation

Ajouter le dépôt officiel Zabbix puis installer les paquets nécessaires :

bash
# Ajouter le dépôt Zabbix root@debian:~# wget https://repo.zabbix.com/zabbix/7.0/debian/pool/main/z/zabbix-release/zabbix-release_latest+debian12_all.deb root@debian:~# dpkg -i zabbix-release_latest+debian12_all.deb root@debian:~# apt update # Installer Zabbix server, frontend et agent root@debian:~# apt install zabbix-server-mysql zabbix-frontend-php zabbix-apache-conf zabbix-sql-scripts zabbix-agent -y
04

Configuration

Démarrer et activer les services Zabbix :

bash
root@debian:~# systemctl restart zabbix-server zabbix-agent apache2 root@debian:~# systemctl enable zabbix-server zabbix-agent apache2

L'interface web est ensuite accessible à l'adresse :

url
http://192.168.2.18/zabbix
05

Vérification

Vérifier que les services sont bien actifs :

bash
root@debian:~# systemctl status zabbix-server ● zabbix-server.service - Zabbix Server Active: active (running)
✅

Le serveur Zabbix est opérationnel. Connectez-vous à l'interface web avec les identifiants Admin / zabbix.
